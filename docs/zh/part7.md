# 第七部分：练习与进步

## 7.1 如何安排一次练习 {#s7-1}

富有成效的个人练习是有结构的。以下模板适合大多数认真学习的歌者，可根据自身需求和老师的建议灵活调整。

**45–60 分钟的练习安排：**

1. **身体热身（3–5 分钟）：** 拉伸，颈部和肩部放松，下颌和舌头拉伸。

2. **气息热身（2–3 分钟）：** "Sss"延音，唇颤音（lip trill）专注于气息流动而非音高。

3. **嗓音热身（10–15 分钟）：** 唇颤音贯穿整个音域，滑音练习，不同元音的音阶练习，在"nay"上做八度滑音，起音练习。（参见第 1.5 节。）

4. **技术训练（15–20 分钟）：** 一两个具体的技术重点——例如混声练习、鸣音（twang）在不同元音间的保持、元音调整练习。*这是整个练习中最有价值的部分，必须做到刻意练习。* 不要只是"热热嗓子然后唱歌"——那样只会强化已有的习惯，而不会建立新的能力。

5. **曲目练习（10–15 分钟）：** 正在排练的歌曲，专注于具体的难点片段，而不只是从头唱到尾。

6. **整理活动（2–3 分钟）：** 轻柔的唇颤音或"ng"音，在中音区做下行滑音。让嗓音回到说话时的协调状态。

### 常见的练习误区

- **跳过热身。** 冷嗓直接唱歌，长期下来会损伤嗓音。
- **没有技术重点。** 只是"把歌唱完"只会强化你已有的模式——包括那些错误的习惯。
- **练习时间过长。** 边际收益下降得很快。45–60 分钟的专注练习胜过 2 小时漫无目的的练习。尤其当你感到疲惫或嗓音已经疲劳时，*停下来*。
- **跳过整理活动。** 常常被忽略，但对嗓音的长期健康很重要。
- **只练自己已经擅长的东西。** 这样无法提升技能。至少花一部分时间在自己能力的边界地带练习。

## 7.2 每日、每周、每月的练习节奏 {#s7-2}

长期的技术进步需要不同时间尺度的规律。

**每日（尽量做到）：**
- 即使在非正式练习日，也做短暂的嗓音热身（10 分钟即可保持协调状态）
- 身体和气息练习
- 一个小的技术重点

**每周：**
- 3–5 次 45–60 分钟的专注练习
- 与老师上课（通常每周一次或每两周一次）
- 至少录一次音并仔细回听

**每月：**
- 重新审视自己的目标和当前状态
- 记录具体技术技能的进步（混声、鸣音、元音调整、气息耐力）
- 根据哪些方面在进步、哪些还没有进步来调整练习重点

**每季度（每 2–3 个月）：**
- 退一步整体评估。比 3 个月前有什么不同？
- 与老师一起确定下一阶段的重点。
- 注意是否在某处卡壳——有时候需要换一个练习方式或角度。

## 7.3 录音与回听 {#s7-3}

你无法在实时演唱时准确听到自己的声音。你脑海中听到的声音——通过骨传导和内部共鸣——并不是听众所听到的声音。要想进步，必须录下自己的声音。

**简单的工作流程：**

1. 使用手机的录音备忘录（Voice Memo）应用或任何基础录音软件。
2. 录下练习过程，尤其是技术训练和曲目练习部分。
3. 第二天再回听（留一点时间距离有帮助；立刻回放往往会引发过度的自我批评，从而扭曲感知）。
4. 重点听：音准、元音质量、声区过渡、强弱变化、气息声是否过重、音色一致性。
5. 挑一件事，在下次练习中重点处理。

**这个过程会让人不舒服。** 大多数歌手都讨厌听自己的录音。这是正常反应，克服它。这种不适感主要来自你习惯的内部声音与实际录音声音之间的落差。坚持规律回听几周之后，你会逐渐校准，觉得有用而不是令人难以接受。

## 7.4 进步的信号 {#s7-4}

演唱技巧的进步是缓慢的、非线性的，而且往往在发生时你自己感知不到。可以留意以下迹象：

**短期（数周）：**
- 特定练习感觉变轻松了
- 气息不那么容易用尽，能撑完更长的乐句
- 特定的高音感觉不那么费力
- 一个之前困惑的概念突然想通了

**中期（数月）：**
- 音域扩展（顶端或底端能稳定发出新的音）
- 混声的协调性更加可靠
- 穿越换声区（passaggio）时音色更加统一
- 曾经靠运气的东西，现在可以随时稳定做出来

**长期（数年）：**
- 形成可辨识的个人声音
- 建立起可随意调用的风格化词汇
- 应对完整演出或整场演唱所需的耐力
- 在高强度演唱负荷下保持嗓音健康

## 7.5 问题出现的信号 {#s7-5}

以下迹象说明出了问题，需要向老师反映（在严重情况下应就诊嗓音科医生）：

- **嗓音疲劳延续到第二天**——正常情况下睡一觉就应该恢复
- **声音嘶哑持续数天以上**，而非由疾病引起
- **高音区消失**，且休息后不能恢复
- **演唱时或演唱后感到疼痛**——唱歌不应该疼
- **突然无法发出某些以前能唱的音**
- **演唱中途声音持续性中断或脱落**

这些问题可能有很多原因——有时是轻微的（过敏、胃酸反流、暂时性疲劳），有时是严重的（声带病变、声带出血、麻痹）。不要忽视。嗓音科医生（专攻嗓音的耳鼻喉科医生）可以用内窥镜检查你的声带，告诉你究竟发生了什么。很多城市都有专门为演唱者服务的嗓音专科诊所。
