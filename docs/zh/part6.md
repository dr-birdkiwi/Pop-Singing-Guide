# 第六部分：麦克风

一个并不总是显而易见的事实：**在流行音乐中，麦克风是你乐器的一部分。** 不是事后的补充，不是录音工具——而是一个从根本上塑造你声音的声学组件，与你的声道同等重要。

流行音乐是伴随麦克风一同发展起来的。麦克风出现之前的声乐传统（歌剧、美声唱法、传统音乐剧）必须在没有扩音的情况下让声音穿透整个乐队。流行音乐从来不必如此。那种亲密感、对话感、需要时的气息感、细腻的强弱变化——这一切审美的存在，正是因为麦克风使之成为可能。

如果你唱流行歌曲却不思考麦克风的问题，你就遗漏了技术中的一个重要环节。

## 6.1 近讲效应 {#s6-1}

大多数现场演出麦克风（以及许多录音室麦克风）都是**心形指向（cardioid）**的——主要拾取正前方的声音，抑制来自后方的声音。心形指向麦克风具有一种叫做**近讲效应**的特性：当你非常靠近麦克风演唱（大约2.5至5厘米以内）时，它们会显著提升低频内容——低音区通常提升6分贝甚至更多。

实际意义是：

- 靠近麦克风演唱会让你的声音听起来更温暖、更饱满、更有分量
- 距离麦克风较远时，声音会更明亮、更清瘦
- 你可以*刻意*利用近讲效应——在低沉/亲密的段落靠近以增加温暖感，在更响亮/明亮的段落后退以获得清晰度

这就是为什么流行歌手不需要用喉咙去制造"胸声的深沉感"。如果他们善用麦克风，麦克风会免费送上低频分量。

## 6.2 距离调控 {#s6-2}

最重要的麦克风技术技能只有一个：**通过移动麦克风来管理动态。**

- **轻声、亲密的段落：** 麦克风距嘴约2.5至5厘米，通常略微偏轴以减少齿擦音和气息声
- **对话式、中等动态的演唱：** 麦克风距离约8至15厘米
- **强力演唱（belt）或高音量段落：** 麦克风距离15至30厘米（最响亮的峰值时可以更远）

这不仅仅是为了避免失真——更是为了控制动态范围，让听众按你的意图感受到你的强弱变化。如果你以同样的麦克风距离演唱一段轻声的主歌和一段强力演唱的副歌，副歌会完全压倒轻声部分。在更响亮的音符时将麦克风拉远，可以保持动态平衡。

观察任何一位技术娴熟的现场流行演出者——他们无时无刻不在调整麦克风距离。这是一项需要习得的技能，应该成为自然而然的反应。

## 6.3 麦克风角度与偏轴技术 {#s6-3}

正对麦克风正面演唱（正轴）信号最强，但也会拾取最多的爆破音（P、B音）和齿擦音（S、Z音）。

对于高音量或多齿擦音的时刻，**将麦克风略微偏轴**——不是正对嘴巴，而是偏移15至30度——可以减少这些问题，同时仍能清晰地捕捉你的声音。

歌手也常常*稍微侧转头部*，或从麦克风顶部斜着唱（而不是直接对着它），来处理高强度的时刻。

## 6.4 现场与录音室麦克风技术 {#s6-4}

这是一个很深的话题，以下是快速的方向性介绍：

**现场演出麦克风**通常是动圈麦克风（Shure SM58、Sennheiser e835等）——坚固耐用，细节捕捉较少，设计用于抑制啸叫。它们适合更近的麦克风技术。现场演出还涉及返送音箱、舞台音量和啸叫管理——这是一套需要多年才能发展出来的完整技艺。

**录音室麦克风**通常是电容麦克风——灵敏度更高，细节更丰富，对技术的要求也更苛刻。细小的失误（口腔噪音、气息声、爆破音）会变得非常明显。录音室演唱通常需要防喷罩（你和麦克风之间的那块屏障）以及稍大一些的距离。

如果你正在跟老师上课，可以询问他们能否在课程中加入麦克风训练。许多受过当代商业音乐（CCM）训练的老师会这样做，有些则不会。麦克风技术确实是独立于声乐技术的一门技能，如果你打算进行演出，值得进行刻意练习。
