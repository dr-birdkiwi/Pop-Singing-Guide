# 第二部分：流行音乐的声音

## 2.1 核心悖论 {#s2-1}

流行唱法的核心真理：

**流行音乐是一种建立在非说话式技术工作之上的、强化说话感的美学。**

它的范本不是被动的咖啡馆闲聊，而是*有能量、有意图的语言*：在停车场对面向朋友大喊，情绪化的告白，开怀大笑，激烈的抱怨，哭泣。那种说话方式本身就已经具备强烈的声带闭合、积极的气息参与和向前的声场存在感。流行音乐将其放大并持续化。

对听众而言，声音应该*感觉*像是随口而出的、充满个性的——但要在巡演中可靠地发出这种声音，在90分钟的演出中坚持下来，需要其下层有奥运级别的训练支撑。那些拥有30年职业生涯的流行歌手（Pink、Beyoncé、Bruno Mars、Stevie Wonder、全盛时期的Mariah）无一不有严谨的技术底层。那些在28岁就嗓子垮掉的人，通常没有。

**刻苦训练，然后藏起训练的痕迹。** 训练的痕迹应该从声音中消失，不应该从练习室里消失。

这是整本指南中最重要的视角框架。其他所有技术要素都服务于这个悖论。

## 2.2 扩音筒——空间、喉位与下颌 {#s2-2}

流行音乐制作有一种独特的声学形态，与美声唱法不同。

**美声唱法**使用我们姑且称之为*大教堂*的形态：喉咙内部高而竖向的空间，喉部下压，软腭高抬，声带上方有大量竖向空间。这产生了圆润、暗沉、温暖的美声音色，无需扩音就能在管弦乐团上方投射。

**流行唱法**使用我们姑且称之为*扩音筒*（或号角）的形态：更宽阔的横向咽腔空间，喉部保持中立或适度上抬，声音收束成一道向前聚焦的声束。这产生了更明亮、更接近说话感的音色，能与麦克风协同工作，而非对抗。

### 空间：形态不同，不是更小

从美声思维转向流行思维时，一个常见误解是：认为"比美声空间小"意味着"喉咙收缩"。并非如此。流行强力演唱者使用的内部空间是巨大的——听听全力演唱的Kelly Clarkson、Beyoncé或Bruno Mars。空间只是*横向*而非*竖向*地展开。宽阔的咽腔、自由的喉咙，收束成向前聚焦的声束。

喉咙收缩是任何风格演唱的敌人。问题不在于"空间更多还是更少"，而在于"什么形状的空间"。

### 喉位：可以移动，不是被钉死

美声唱法中，你可能听说过"保持喉位低"，或见过旨在在整个音域内维持低喉位的练习。在流行唱法中，这通常不是目标。

流行唱法的喉位在低音区和中音区保持在**说话时的自然高度**，随着音高上升——尤其在强力演唱（belt）和混声中——会**适度上抬**。这是正常的、健康的，也是这个风格声音所必需的。

你要避免的是*张力驱动*的上移——喉部因喉咙挤压而向上猛冲。随着健康的音高上升而自然产生的上移是没问题的。来自喉咙代偿的被迫上移才是问题。

如何区分：
- 自然上移：音色自由，无挤压感，可持续
- 被迫上移：音色收窄刺耳，有掐脖子的感觉，令人疲惫

### 下颌：放松但有参与感

下颌是流行技术中最常被误解的要素之一。

"放松"不意味着"松垮"或"软塌"。一个完全放松、随意摆动的下颌会破坏高音稳定所需的共振调谐，让你的音色失去清晰度。下颌需要**关节处不紧绷，但牢固保持在下放的姿势**。

把它想象成握网球拍时的松握：不夹紧，不软塌，有参与感但不抓死。

尤其在高音时，你的下颌应该打开并*保持*在那里——由打开下颌的肌肉主动持住这个位置，而不是靠闭合下颌的肌肉施力。如果你的下颌在持续高音时摇晃、漂移或合拢，你将失去声学稳定性，那个音听起来会散焦、不聚。

### 舌头

放松，向前，以说话时的自然方式塑造元音。舌尖默认靠近下齿背面，但可以自由移动来塑造元音。不要试图压住它或向后拉。美声教学中常见的提示（"把舌头压平"）往往会导致舌根下压，产生闷头、人工感的音色——在流行唱法中要避免这种做法。

流行唱法中最大的舌头问题是**舌根张力**——舌根向下夹紧。你可以用手指感受下颌下方、下巴正后方。如果那个区域在你唱歌时硬如石头，说明你带着舌头张力。练习前做哈欠、叹气和温和的舌头伸展，有助于释放这种张力。

### 嘴唇和脸颊：微笑陷阱

嘴唇是你扩音筒喇叭口的一部分。它们得到的关注少于下颌和舌头，但有一条流传甚广的民间流行教学建议会真实地破坏高音：

**"微笑，让你的声音更明亮、更靠前。"**

*轻柔的、面颊内侧上提*——那种你真正开心或准备迎接某人时脸上自然带有的提起、警觉感——是没问题的，有时也有用。它能鼓励软腭参与，以及上面部的一种开放感。

*宽大的、横向拉开的微笑*——嘴角向后拉、露出牙齿——是有破坏性的。它会：

- 通过向后拉嘴角来缩短声道
- 通过相连的肌群将喉部向上牵拉
- 以一种对抗元音调整的方式展宽元音
- 容易使音色变薄，产生下一节所警告的"破散"音色

如果有人告诉你"唱歌时要微笑"，请检查这个提示实际上要你做什么。面颊内侧小幅上提 = 没问题。横向拉开的大笑 = 问题。尤其在高音时，你的口型通常应该是*高过宽*的（更多竖向开口，更少横向拉伸）——这与拉开的微笑恰恰相反。

## 2.3 元音——感知上像说话，执行上经过调整 {#s2-3}

元音是流行技术的核心所在。基本原则：

**在低音区和中音区，唱元音的方式接近说话时的方式。高过换声区之后，元音必须经过微妙的调整。**

### 为什么调整是必要的

随着音高升高，你的声道声学属性需要与你所发出的音高调谐。你在说话音域内使用的元音形态在那里运作良好，但如果原封不动地带入高声区，就会产生声学不匹配（并需要大量气压）。

不做调整会发生什么：
- 声音"炸开"——变得刺耳、单薄
- 喉部向上猛冲来代偿
- 你开始用力
- 最终，嗓子出问题

做了调整之后：
- 元音在声学上与音高对齐
- 喉部保持自由
- 那个音感觉更省力
- 听众仍然能理解歌词

听众能辨认出原来的词，是因为辅音将其框定，也因为调整是微妙的。*听起来*没有被调整过，*实际上*已经被调整过了。

### 两种策略：收窄与展宽

元音调整有两种相反的方向，用于不同目的。

**收窄（默认，可持续）：**
- "Ah"（如"father"中）偏向"uh"（如"love"中）
- "Ee"（如"see"中）偏向"ih"（如"sit"中）
- "Ay"（如"say"中）偏向"eh"（如"set"中）

这是安全、可持续的路线。它将元音调谐为滑入平衡或头声主导的混声。在上行进入高音区时，将此作为你的默认策略。

**展宽（风险较高，用于呐喊式强力演唱）：**
- "Ah"偏向"æ"（如"cat"中）
- "Eh"偏向"æ"

这会提升第一共振峰，将胸声协调稍微向上延伸一点才发生过渡。它需要更高的气压、更多的支撑，可持续的空间也更短。但这是摇滚和强力流行强力演唱的标配——那种有攻击性、有棱角的胸声强力演唱，正是建立在展宽元音上的。

知道自己在用哪种策略——以及为什么——很重要。默认选择展宽只是因为听起来有力量，却不明白这是高成本的选择，正是歌手们模仿自己喜爱的摇滚强力演唱者而把嗓子搞垮的方式。

### 如何练习元音调整

1. 选一首在你高音区有持续高音的歌曲。
2. 找出落在最高持续音上的歌词。
3. 记下每个词的元音。
4. 在那些音符上，尝试把元音向"uh"方向偏移（收窄）。录音回听。
5. 与唱未调整版本的元音对比。未调整版本通常听起来更刺耳、感觉更难；调整后的版本听起来更顺滑、感觉更省力——但那个词仍然清晰可辨。

这是一项极大受益于老师反馈的内容。"恰当"的调整量是微妙的，因歌手、元音和音高而异。老师能听出你是调整不足（仍在破散）还是调整过度（词语听起来过于失真）。

## 2.4 起音——一个音是怎么开始的 {#s2-4}

开始一个音的方式叫做**起音**，流行唱法使用的起音方式比美声唱法丰富得多。

### 四种主要起音方式

**1. 协调起音（或"balanced"起音）。** 气流和声带闭合同时到达。干净、中性的开始。这是美声的默认方式，也是流行唱法很好的中性默认值。

**2. 气息起音。** 气流在声带完全闭合之前稍早开始流动。起音带有柔和、气息感。用于亲密感、脆弱感、感性感。Ariana Grande的主歌、Billie Eilish的整体美学、Sam Smith的大量作品。

**3. 声门起音（或"硬"起音）。** 声带在气压积聚之前完全闭合，然后以一个小小的"冲劲"释放。用于强调、攻击感、锋利感。"uh-oh"这个声音就以声门塞音开始。Adele在乐句开头用它来表达情感强调。R&B和近嘻哈风格的人声也会风格化地使用。

**4. 气泡音起音。** 音符以气泡音（vocal fry）开始——那种低沉、爆裂的声音——然后过渡到正常发声。这是一种标志性的流行声音（Halsey、早期Ariana、大量当代电台流行）。我们将在5.2节介绍气泡音的技术细节，因为那里有真实的注意事项。

### 规则不是"永远使用协调起音"，而是"有意识，不是偶然"

风格性气息感（stylistic breathiness）没问题——Ariana Grande靠这个建立了整个职业生涯。*无意识的*气息感，也就是因声带闭合不足或气息支撑不够而漏气，则不行：它消耗你的嗓音，造成疲劳，遮盖音乐细节。

声门起音也一样。有意识的强调 = 风格选择。习惯性的声门挤压 = 通向嗓音伤害的路。

### 练习起音

选一个你舒适中音区内的单一持续音，用"啊"连唱三次，每次之间稍作停顿：
- 第一次：协调起音（干净、中性的开始）
- 第二次：气息起音（以轻微叹气开始，然后加入音色）
- 第三次：声门起音（开始时有轻微"冲劲"）

录音回听。你能听出区别吗？你能按需产生每一种起音吗？这种意识是基础——一旦你能有意识地选择起音方式，你就掌握了流行唱法中最具表现力的工具之一。

## 2.5 鸣音——扩音筒的吹嘴 {#s2-5}

如果说流行唱法中有一个技术概念能在最短时间内产生最显著的效果，那就是**鸣音（twang）**。

### 鸣音是什么

鸣音是声带上方一个叫做**杓会厌括约肌**（有时称为会厌上管）的小空间的收窄。当这个空间变窄时，它能做到两件事：

1. 充当一个小型高压喷嘴，提高发声效率
2. 在高频频谱能量上产生强烈的提升——也就是你在流行嗓音中听到的那种"响亮感"或"锋利感"

这种高频提升落在与美声唱法所称的"歌者共鸣峰"重叠——但机制上并不完全相同——的频段内。声学邻域相似；产生路径不同。对你来说重要的是：鸣音是流行声音所具备的那种穿透力、明亮感和锋利感的来源。

### 鸣音是扩音筒的吹嘴

还记得2.2节说的扩音筒形态吗？鸣音就是那个扩音筒底部的窄喷嘴。上方宽阔的咽腔空间汇聚进窄小的杓会厌漏斗，后者充当吹嘴。宽大的喇叭口 + 窄小的喷嘴 = 聚焦、有穿透力的流行声音。

这就是为什么"更宽的空间 + 更多鸣音"并不与"聚焦、向前的声音"相矛盾。它们是同一声学现象的两半。

### 如何找到鸣音

有几种可靠的方法：

**方法一：夸张的鸭子音。** 模仿一只有态度的卡通鸭。或一个讽刺的孩子："yeahhhh, riiiight."  或女巫的笑声："heh heh heh." 那种让人烦躁的、响亮的、几乎刺耳的质感就是夸张的鸣音。一旦你能产生它，再按口味调整回来。

**方法二："apple"中的/æ/元音。** 夸张地说"apple"："AP-ple." /æ/元音天然会收窄杓会厌空间——它替你做了大量工作。只持续发开头的元音："æææ." 在一个舒适的音高上保持那个明亮的元音。

**方法三：呼喊。** 想象在停车场对面向朋友喊："Hey! HEY!" 那种自然的呼喊本身就内含鸣音——这是人声在没有扩音的情况下跨越距离传递的方式。

### 怎么知道自己做对了

- **感觉毫不费力，而非费力。** 正确的鸣音感觉是声音变得更亮、更响——*而不是*你在更努力地工作。如果你在用喉咙或胸部向外推来获得更多音量，那是张力，不是鸣音。
- **感觉像一个小空间，高而靠前。** 几乎在鼻子和上齿后面。下方的喉咙应该感觉开放而放松。
- **音准变得更稳定。** 鸣音会锁定音准。如果增加"锋利感"让你偏高或偏低，你可能是在挤压喉部，而不是在收窄会厌上管漏斗。
- **你可以持续使用而不疲劳。** 真正的鸣音是嗓音能做到的最高效的事情之一。如果唱20分钟后你的嗓子就累了，说明你是在用喉咙挤压代替鸣音。
- **它与鼻音是可分离的。** 捏住鼻子同时发鸣音。声音不应该有太大变化。如果捏住鼻子后声音完全变哑，说明你是在把声音导向鼻腔（鼻音），而不是在收窄杓会厌括约肌（鸣音）。这是不同的机制。

### 鸣音与鼻音

这一点至关重要：**鸣音和鼻音是不同的东西。**

- **鸣音** = 喉咙内的收窄（杓会厌括约肌），位于声带上方
- **鼻音** = 软腭下落，让一部分气流和声音进入鼻腔

在未经训练的歌手中，它们经常同时出现（同样的"整体收紧"趋势往往同时产生两者），但它们是有各自解决方案的不同机制：

- 未锚定的鸣音会变得*刺耳*。解决方法：锚定它（见下文）。
- 软腭完全塌陷会产生完全的*鼻音*——闷、嗡嗡作响，遮盖底层音色。解决方法：重新激活软腭（想象哈欠开始时的感觉）。

不要把"向前的声场"和"用鼻子唱"混淆。它们感觉相似，但声学上和生理上是不同的。

**一个重要的细节：鼻音是个旋钮，不是开关。** 喉咙和鼻腔之间的开口（腭咽口）可以完全关闭、完全打开，或介于两者之间的任何程度。完全塌陷——浓重、嗡嗡的鼻音——在几乎任何情境下都不受欢迎。但*受控的、轻微的鼻腔漏气*是真实存在且被大量使用的流行和R&B色彩：它产生的那种"哭腔"、"诉苦感"或"哀求感"，在Justin Bieber、Shawn Mendes的声音里、在大量乡村跨界流行中，以及在Ariana Grande某些情感时刻里都能听到。少量鼻音叠加在清晰的底层音色之上，能增加情感颗粒感而不遮盖声音。

需要持有的框架：
- **无意识的完全鼻音** = 瑕疵（闷哑、嗡嗡，遮盖你的音色）
- **受控的轻微鼻腔漏气** = 风格色彩（增加哭腔/诉苦感，有意使用）
- **完全封闭** = 干净音色，无鼻腔着色

这是最好与你的老师一起培养的能力之一——他们能听出你的鼻音是有意识的、在风格上有效的，还是无意识的、在模糊你的声音。

### 锚定：为什么鸣音必须有支撑

鸣音常被称为"流行唱法的作弊码"，因为它效率极高——无需更多气压就能大幅增加明亮感和穿透力。但这种高效性在孤立使用时也带来了危险。学了鸣音但没学会锚定的歌手，发出的声音刺耳、尖细、娇气，而不是有力量。

**鸣音必须有身体锚定（anchoring）作为支撑：**

1. **气息/核心支撑**——稳定、有支撑的声门下气压（我们将在第四部分详细讲解）
2. **放松但有参与感的下颌和舌根**——上方明亮的漏斗无法与下方的张力共存
3. **下方一些胸腔/身体共鸣**——给音色增添厚重感，而不只是锋利感

没有锚定的鸣音 = 哨声。有锚定的鸣音 = 小号声。

### 练习鸣音

找到鸣音之后，工作是**把它带过各种元音**。鸣音天然存在于/æ/（apple、cat）和/ɛ/（red）中。它天然抗拒圆唇元音——/o/（go）、/u/（you）、/ɔ/（caught）。

一个实用的日常练习：

1. 在舒适的中音区音高上，用/æ/找到你的鸣音。保持它。
2. 缓慢地将元音沿以下序列迁移，尝试保持同样的响亮质感：
   /æ/ → /ɛ/ → /ɑ/ → /ɔ/ → /o/ → /u/
   （apple → bed → father → bought → go → you）
3. 最后两个会"反抗"你。这正是学习发生的地方。
4. 在几个不同的音高上重复。

每天做这个练习五分钟，持续几周，将会显著扩展你对流行声音的控制力。
