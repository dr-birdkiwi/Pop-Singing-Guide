# 附录：术语表

**Anchoring（身体锚定）** — 躯干、颈部和发音器官的肌肉稳定性，在强力或持续演唱时为声音提供支撑。

**Aryepiglottic sphincter / epilaryngeal tube（杓会厌括约肌 / 喉上管）** — 位于声带正上方的狭小空间。收窄此处可产生鸣音（twang）。

**Belt（强力演唱）** — 在流行音乐和音乐剧中用于高音区的响亮、厚实、类胸声音色。对大多数歌手而言，高音区的强力演唱在声学上是以头声混声（CT肌主导）来模拟胸声的效果。

**Break（换声点）** — 在未经训练的声音中，胸声与头声之间在换声区（passaggio）上方发生的可闻翻转。通过发展混声（mix）可以使其平滑过渡。

**Cardioid（心形指向）** — 麦克风的拾音极坐标模式，拾取正面声音并抑制背面声音。大多数现场演出麦克风采用心形指向。

**Chest voice（胸声）** — 低沉、有分量、类说话感的声区。甲杓肌（TA肌）主导。共鸣感受于胸腔和喉部。

**Cricothyroid / CT（环甲肌）** — 使声带拉长变薄的肌肉。环甲肌（CT肌）主导的协调方式产生头声。

**Falsetto（假声）** — 较轻、通常偏气息感的高声区，尤多见于男声。与头声有重叠。

**Formant（共振峰）** — 声道中的共鸣峰值。不同元音由各自的共振峰模式加以区分。

**Head voice（头声）** — 较高、较轻的声区。环甲肌（CT肌）主导。共鸣感受于面部和头部。

**Larynx（喉）** — 喉部，即俗称的"嗓子眼"，内含声带。

**Mix（混声）** — 将胸声与头声特征融合的协调方式，使歌手能够平滑地穿越换声区（passaggio）演唱。可偏胸声、平衡或偏头声。

**Onset（起音）** — 一个音符开始发声的方式。类型包括：协调起音、气息起音、声门起音和气泡音起音。

**Passaggio** — 意大利语，意为"通道"。声区之间的过渡区域。上换声区（胸声与头声之间）是流行歌手最需要关注的实践要点。

**Phonation（发声）** — 声带产生声音的过程。

**Proximity effect（近讲效应）** — 靠近心形指向麦克风演唱时产生的低频增强效果。

**Resonance（共鸣）** — 声道对声带振动的放大与塑形。

**Singer's formant（歌手共振峰）** — 美声唱法训练有素的歌手所具备的高频共振峰聚集，使声音能够不借助扩音而投射。流行音乐的"Ring"音色与此有重叠，但并不完全相同。

**Subglottal pressure（声门下气压）** — 声带下方的气压。强力演唱（belt）时气压高；气泡音（fry）时气压低。

**Support（气息支撑）** — 躯干肌肉的协调运作，用于控制演唱时的呼气节奏。

**Thyroarytenoid / TA（甲杓肌）** — 使声带增厚缩短的肌肉。甲杓肌（TA肌）主导的协调方式产生胸声。

**Tessitura（最适音区）** — 声音最舒适的音高范围，有别于总体音域（range）。

**Twang（鸣音）** — 杓会厌括约肌收窄所产生的高频频谱增强效果。即流行唱法中声音的"Ring"感或"Edge"感。

**Vibrato（颤音）** — 持续音上的音高周期性振荡，健康状态下通常为每秒 5–7 次。

**Vocal fold / vocal cord（声带）** — 喉部的两片组织，振动产生声音。

**Vocal fry（气泡音）** — 低沉、嘎吱感的声区。声带松弛且厚，气压极低。

**Vocal tract（声道）** — 声带上方的咽腔、口腔和鼻腔。将声带的原始振动塑形为可听见的声音。

**Whistle register（哨音声区）** — 头声之上的最高声区，部分女高音用于演唱极高音符。
