# 如何使用本指南

本指南是一份参考文档，不是一门课程。内容按主题组织，你可以直接跳到与当下状态最相关的章节。有些部分会立刻读懂；另一些需要经过数周乃至数月的练习才能真正理解。这很正常——回来重读就好。

在整个学习过程中，请牢记以下几个原则：

**解剖学是用来诊断的，不是用来实时控制的。** 了解高音区强力演唱（belt）涉及特定的肌肉协调，有助于你理解为什么某些方法行不通、另一些方法却有效。但这*不能*帮助你在演唱时实时找到那个音。演唱是一种由音效目标、元音感觉和情感意图驱动的自动化运动技能——而不是靠有意识的肌肉指令来完成的。本指南中的解剖学知识是用来帮你选择练习方式、诊断问题所在的，不是用来在演唱乐句中途靠脑子去控制的。

**以音效目标代替肌肉目标。** 练习时，瞄准一个目标音色或感觉，而不是某个具体的肌肉动作。"更明亮"、"这里多一些嗡鸣感"、"像向远处的朋友喊话一样"——这些提示有效。"更多地激活你的环甲肌（CT肌）"则不然。

**大多数规则都有反规则。** 流行演唱技术是一套条件性协调的系统。元音可以收窄，*也*可以扩宽，取决于你想要的混声效果。颤音可以用，*也*可以不用，取决于乐句的需要。气压可以很高，*也*可以接近于零，取决于你在强力演唱还是使用气泡音。中级歌者学规则，进阶歌者学每条规则在何种条件下会反转。本指南力图同时传授这两个层面。

**范围说明。** 本指南针对当代英美流行音乐、R&B 及强力演唱（belt）传统。K-pop、拉丁流行、乡村、传统音乐剧、爵士，以及独立/民谣流行与本指南内容有所交叉，但在某些方面各有不同的声音制作惯例。如果你涉足这些流派，本指南中部分建议需要相应调整。

**你的老师是第一资源，本指南是辅助。** 文字能够描述发生了什么以及为什么——但文字听不见你唱歌，看不到你的下颌动作，也感受不到你的紧张。本指南会全程标注哪些问题真正需要老师的耳朵来判断，哪些你可以自己高效地练习。
