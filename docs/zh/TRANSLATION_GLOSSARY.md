# 翻译术语表 — Translation Glossary

This is the canonical reference for all translation agents working on this project.
**Do not deviate from these terms.** If a term is missing, use the closest established Chinese vocal pedagogy term, note the English in parentheses on first use, and flag it for review.

---

## Translation Principles

1. **Tone**: The original is authoritative, direct, and friendly — like a knowledgeable teacher writing notes. Match this in Chinese. Not academic/stiff (避免学术腔), not casual-internet. Clear, confident prose.
2. **Terms with no clean Chinese equivalent**: Use the English word directly in the Chinese text. Do NOT force an awkward Chinese coinage. For example, "twang" or "belt" can appear as-is if the Chinese equivalent sounds unnatural in context.
3. **Terms with good Chinese equivalents**: Use the Chinese term. On first occurrence in each file, you may optionally add the English in parentheses: 颤音（vibrato）.
4. **Headings**: Keep all `{#s1-1}` style anchor IDs unchanged. They are language-neutral and required for sidebar navigation.
5. **Links**: Do not change any link paths. Keep all `/part1`, `/part2`, etc. links as-is.
6. **Structure**: Preserve all markdown formatting — bold, italics, bullet lists, numbered lists, code blocks, blockquotes, tables.
7. **Section intro callouts**: Preserve all `:::` admonition blocks if any exist.
8. **Do not translate**: GitHub usernames, URLs, file paths, technical IDs.
